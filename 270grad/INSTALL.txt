# Installation:

* pyOSC, uses socket, to install, simply run, $ sudo ./setup.py install, Link: https://trac.v2.nl/wiki/pyOSC
* simpleOSC, wrapper around pyOSC, to install run $ sudo python setup.py install, Link: http://www.ixi-audio.net/content/body_backyard_python.html

* all gstreamer stuff
* python stuff

* python-gst0.10, python-gst0.10-dev, python-gtk2, python-gobject, python < v2.6

* v4l2loopback, use github repo from IOhannes, git://github.com/umlaeute/v4l2loopback.git
  * when loaded # of devices have to be defined i.e. modprobe v4l2loopback devices=10
* gst-v4l2loopback, https://github.com/umlaeute/gst-v4l2loopback/
  + dh-autoreconf module-assistant debhelper mercurial mercurial-git

* test gst-loopback: 
    gst-launch v4l2src ! videoscale ! videorate ! video/x-raw-yuv,width=640,height=480,framerate=25/1 ! ffmpegcolorspace ! v4l2loopback
    gst-launch v4l2src device=/dev/video1 ! autovideosink
